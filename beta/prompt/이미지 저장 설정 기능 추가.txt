- 이미지 저장 버튼 클릭 시 모달 창이 떠서 아래 설정이 가능하다.
	- 타이틀 표시 토글 (기본 on)
	- 버리기 및 변경 카드 표시 토글 (기본 on)
		- drop 및 changes 카드들을 제외함
	- 포인트 표시 토글 (기본 on)
		- 총합 포인트, 카드 별 포인트, 유닛 별 총합 포인트 표시 여부 설정
		- 하부 옵션을 제공. '포인트 표시' 토글을 off로 할 시 하부 옵션들이 모두 off가 됨
			- 총합 포인트 표시
			- 카드 포인트 표시
			- 유닛 총합 포인트 표시
	- 서브 카드 표시 토글 (기본 on)
		- 최하단에 표시되는 서브 카드들을 표시할지에 대한 여부 설정
	- (hidden 카드 보유 시) 숨김 카드 공개 (기본 on)
		- 이 기능은 현재 팝업창에서 확인하고 있는데, 해당 기능을 이 옵션으로 옮겨야 함
